## Dash

### create and activate virtual environment

`python3 -m venv ./venv `
` source ./venv/bin/activate`


### necessary modules

`pip install dash`

`pip install pandas`

`pip install pyyaml`

`pip install psycopg2-binary`

###
starting a local dash server.
using
`python -m src.dash.app`
in the root directory.
